# JFrog Runtime Helm Chart

## Chart Details
* Installs the [JFrog Runtime](https://jfrog.com/runtime/) service. 
* Optionally deploys PostgreSQL.
  * NOTE: For production grade installations it is recommended to use an external PostgreSQL.

## Prerequisites
* JFrog Platform installed with Artifactory and Xray running.

## Installation Guide
See the official [JFrog Runtime installation guide](https://jfrog.com/help/r/jfrog-installation-setup-documentation/installing-jfrog-runtime-security) for detailed instructions on how to install JFrog Runtime using this Helm chart.
